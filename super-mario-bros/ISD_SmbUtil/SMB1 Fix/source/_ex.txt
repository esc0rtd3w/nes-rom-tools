-----------------------------------------------
  拡張子操作モジュール  ver 0.50  by わいわい
-----------------------------------------------

■概要
  Hot Soup Processorのファイル保存ダイアログは、拡張子を省略しても
  拡張子は追加されません。これって結構痛いんですよね。そこで、この
  [拡張子を追加する]命令を使うと保存ダイアログの操作性が向上します。
  [拡張子を取得する]命令はファイル判別に、[拡張子を削除する]命令は
  同名で別の拡張子のファイルを探す時に役に立ちます。
  ということで、このモジュールも少しは役に立つと思います。
  いくつかの命令はサブルーチン版も同梱してます。

■命令

  □拡張子を追加する
    (既に拡張子がついている場合は追加しません)

    pex p1,p2
	p1 : 拡張子を追加したい文字列変数
	p2 : 追加する拡張子の文字列・文字列変数

	stat	0 : 成功
		1 : 失敗(パラメータが省略された)


  □拡張子を取得する
    p1に拡張子を、拡張子が無い時は""、ディレクトリは"\\"を代入します。

    gex p1,p2
	p1 : 取得した拡張子を格納する文字列変数
	p2 : 拡張子を取得したい文字列変数

	stat	0 : 成功
		1 : 失敗


  □拡張子を削除する
    p1のファイル名から拡張子だけを消します。

    dex p1
	p1 : 拡張子を削除したい文字列変数

	stat	0 : 成功
		1 : 失敗


  あとは、付属のサンプルスクリプトを参照してください。

■注意事項
  付属のモジュール・スクリプトの著作権は"わいわい"にあります。
  フリーウェアで、内容を改変しない限り、転載の連絡も不要です。
  このモジュールを組込んだプログラムも自由に配布して構いません。

  当然ワイルドカード"*"や"?"を含む拡張子は追加できません。
  拡張子に全角文字を使うプログラムを作るのも止めましょう。

  このモジュールを利用するには、モジュール機能に対応したHSPver2.5以上が必要です。
  モジュールを使うためにスクリプトに[ #include "_ex.as" ]を記述してください。
