Keyboard Shortcuts

Selection
Tab Select next object or enemy
Shift+Tab Select previous object or enemy
Control+Tab Switch between object or enemy selection
Object editing
Arrows Move selected item
Space Change palette
P Change palette
+ Next object/enemy type
- Previous object/enemy type
Shift+` Select palette 0
Shift+1 Select palette 1
Shift+2 Select palette 2
Shift+3 Select palette 3
0-9, A-F Type in object index (instead of using + and -)
Home Bring to front
End Send to back
Backspace, Delete Delete selected object/enemy
Screen Editing
Page Up Previous screen
Page Down Next screen
< Add object
> Add enemy
Map
Shift+ Arrows Navigate Map
Numpad-0 Delete current screen
Numpad-1 to Numpad 5 Set level for screen
